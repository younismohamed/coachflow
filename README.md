# CoachFlow - Employee Coaching Management System

A production-ready SaaS application for managing employee coaching sessions.

## Tech Stack
- **Frontend**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Backend**: Supabase
- **Auth**: Supabase Auth (Google OAuth)
- **Email**: Resend
- **Deployment**: Vercel
